运行时API示例 ​

本页演示了VitePress提供的一些运行时API的用法。

主要的useData() API可用于访问当前页面的站点、主题和页面数据。它可以在.md和.vue文件中使用：

md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

## 结果

### 主题数据
<pre>{{ theme }}</pre>

### 页面数据
<pre>{{ page }}</pre>

### 页面前置数据
<pre>{{ frontmatter }}</pre>
